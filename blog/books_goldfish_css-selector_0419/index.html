04-19 ::placeholder

輸入框內顯示「提醒文字」:
<input type="text"> 使用 placeholder 屬性來設定要顯示給使用者看的文字
輸入框樣式限「字體、背景」有關的 CSS 屬性 :
僅小部份 CSS 屬性可使用，常用有 color、font-size、font-style，更多可參考 ::first-line 允許的屬性值

結構

...
<section>
  <fieldset>
    ...
    <h2><input class="input-txt" type="text" placeholder="請輸入英文 → Enter" ... onkeypress="…"></h2>  
    ...
  </fieldset>
</section>
...

樣式

…
.input-txt{
  line-height: 2;
}
input::placeholder {
  color: var(--black-50);
  background: var(--black-20);
  padding: 10px 0; /*解決輸入框的多餘白邊*/
}
