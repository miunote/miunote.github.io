刻意練習

圓點 雨滴 半圓 弧線．pattern

By Miu 30 July 2021 Graphic

1-001 polka dot

圓點: 替換圖案
雨滴: 加入圖案
半圓: 刪除圖案
弧線: 旋轉圖案

圓點

間隔一個距離，把圓點替換成不同(花)圖案，會有不同的情境感。

雨滴

在中心點加入一個具體且相關的(雨傘)主要圖案，呈現出更明確的主題。

半圓

刪除部份圖案，刻畫出想要的造形。
( 字母 M )

( 向上擴散 )

弧線

旋轉不同角度疊加上去，會有意想不到的新圖案產生。
( 45度 )

( 90度 )
